SmartCRM

SmartCRM is a full-stack Customer Relationship Management (CRM) platform designed to streamline lead management, customer tracking, user administration, and sales analytics.

Features
JWT-based Authentication & Authorization
Role-Based Access Control (Admin, Manager, Sales)
Dashboard with Real-Time Business Metrics
Lead Management (Create, Update, Delete, Assign)
Lead Status Pipeline Tracking
Customer Management
User Management
Sales Team Performance Analytics
Search Functionality
Responsive Modern UI
